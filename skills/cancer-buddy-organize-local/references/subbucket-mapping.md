# Sub-Bucket Mapping — cancer-buddy-organize-local

> 11-bucket taxonomy 的子桶映射。Layer 2 (Claude vision 分类) 必须把每个文件落到具体子桶，不能停在桶级。

## 完整桶 + 子桶清单

```
01_当前状态/                          # 由 Layer 3 自动生成的 当前状态.md，不放原始文件
│   └── 历史快照/                     # 每次 organize 重跑前的旧版当前状态备份

02_诊断与分期/
│   └── 病理报告/                     # 完整病理诊断 (含大体描述 + 镜下描述 + 病理诊断)

03_分子病理/
│   ├── 基因检测/                     # NGS / WES / panel 报告
│   ├── 免疫组化/                     # 单独 IHC 报告 (PD-L1, HER2, Ki-67, MMR 等)
│   └── HPV 分型/                     # HPV PCR 基因分型 (16/18/31/33/...)

04_影像学/
│   ├── CT/                           # 平扫 / 增强 CT
│   ├── MRI/                          # 平扫 / 增强 / DWI / DCE-MRI
│   ├── PET-CT/                       # PET/CT, PET/MR
│   ├── 超声/                         # B 超 / 彩超 / 心动图 / 阴道镜 / 经阴道超声
│   ├── X光DR/                        # 胸片 / 骨片 / 钼靶 / DR
│   └── 其他/                         # 内镜 / 骨扫描 / 核素显像

05_检验检查/
│   ├── 血常规/                       # 全血细胞 + 五分类
│   ├── 生化肝肾功/                   # 肝功 / 肾功 / 电解质 / 心肌酶 / 血脂 / 血糖
│   ├── 肿瘤标志物/                   # CEA/CA199/AFP/CA125/SCC/HE4/PSA/VEGF
│   ├── 凝血/                         # PT/APTT/D-dimer/纤维蛋白原
│   ├── 免疫/                         # 免疫球蛋白 / 补体 / 自身抗体 / 病毒筛查
│   ├── 淋巴亚群/                     # CD3/CD4/CD8/B细胞/NK
│   ├── 甲功/                         # 甲功五项 / 七项 / 抗体
│   ├── 性激素/                       # FSH/LH/E2/PRL/T/孕酮
│   ├── 心脏标志物/                   # 肌钙蛋白 / BNP / CK-MB
│   └── 其他/                         # 尿常规 / 便常规 / HCG / 血型

06_治疗记录/
│   ├── 化疗/                         # 化疗医嘱 / 出库单 / 化疗执行记录
│   ├── 放疗/                         # 外照射 / 后装腔内放疗 / 立体定向 / 质子
│   ├── 免疫治疗/                     # PD-1/PD-L1/CTLA-4/双抗
│   ├── 靶向/                         # TKI / 单抗 (Her2 / 抗 EGFR 等非免疫的)
│   ├── 手术-内镜/                    # 手术记录 / 麻醉单 / 阴道镜 + 活检 + ECC
│   └── 支持治疗/                     # 升白 / 升板 / 升血红蛋白 / 镇痛 / 止吐 / 营养

07_合并症与用药/                      # 合并症列表 + 长期口服药清单
08_出院小结/
│   └── 入院小结/                     # 入院记录单独子桶 (区别于含全程的出院小结)

09_患者补充/                          # ★ v2 新增
│   ├── manual_timeline/              # 患者/家属手工整理的时间线 (.txt/.md/.docx)
│   ├── wechat/                       # 微信群 / 个人聊天截图与导出
│   ├── voice_transcripts/            # 口述病史 / 与医生通话转录
│   └── handwritten/                  # 患者手写笔记照片 (vs 04 影像学的医院出具材料)

10_原始文件/
│   ├── 原始未遮挡/                   # 字节级镜像 PII 未脱敏，仅本地审计
│   ├── _duplicates/                  # SHA256 同源被去重的副本
│   └── 未分类/                       # Layer 2 confidence < 0.5 的文件 (人工 review)

11_诊断证明/                          # 医院开具的对外证明 (≠ 出院小结)
```

## 子桶判定规则（Layer 2 决策树）

读 `ocr/<basename>.md` 文本后按下列顺序判：

### 步骤 1：粗桶 (01-11)

跳过 01 (synthesized)、09 (curated)、10 (mirror)。从 02-08 + 11 中选 1。

### 步骤 2：细桶（仅当粗桶有子桶时）

#### 04_影像学

| 关键词 | 子桶 |
|---|---|
| 磁共振 / MR / MRI / DWI / DCE | MRI/ |
| PET / 正电子 / SUV | PET-CT/ |
| CT / 计算机断层 / 平扫 / 增强 (无 MR/PET 关键词) | CT/ |
| 超声 / B 超 / 彩超 / 阴道镜 / 经阴道 / 心动图 / 多普勒 | 超声/ |
| X 线 / DR / 胸片 / 钼靶 / 骨片 / 放射 (排除 CT/MR) | X光DR/ |
| 内镜 / 胃镜 / 肠镜 / 支气管镜 / 骨扫描 / 核素 | 其他/ |

#### 05_检验检查

按 OCR 文本中**主要检测项目**的多数派判定：

| 主要项目 | 子桶 |
|---|---|
| WBC + RBC + HGB + PLT 占主体 (CBC) | 血常规/ |
| ALT + AST + 胆红素 + 肌酐 + 电解质 + 白蛋白 + LDH + GGT + 血脂 + 血糖 占主体 | 生化肝肾功/ |
| CEA / CA199 / CA125 / AFP / SCC / HE4 / VEGF / NSE / Cyfra21-1 / proGRP / PSA / 神经特异烯醇酶 | 肿瘤标志物/ |
| PT / INR / APTT / D-dimer / 纤维蛋白原 | 凝血/ |
| 免疫球蛋白 / C3/C4 / 自身抗体 / 病毒五项 / HBsAg/HCV/HIV / TB-spot | 免疫/ |
| CD3/CD4/CD8/B细胞/NK + 计数 | 淋巴亚群/ |
| TSH + FT3 + FT4 + (TGAb/TPOAb) | 甲功/ |
| FSH + LH + E2 + 睾酮 + 孕酮 + PRL + β-HCG | 性激素/ |
| 肌钙蛋白 + BNP + NT-proBNP + CK-MB | 心脏标志物/ |
| 其他（**含 ESR / CRP / Procalcitonin / Vit D / B12 / Folate / HbA1c**） | 其他/ |

**易混淆的 lab 路由 — 必查**（v2.1 新增，针对常见误分类）：

| 项目 | 易误判到 | 正确子桶 | 理由 |
|---|---|---|---|
| **ESR (血沉/红细胞沉降率)** | 血常规/ ❌ | **其他/** ✅ 或 免疫/ | ESR 不是血常规 (CBC) 的一部分；它是非特异性炎症指标，与 CRP/PCT 同类 |
| **CRP (C-反应蛋白)** | 生化肝肾功/ ❌ | **其他/** ✅ 或 免疫/ | 急性期反应物，不是肝肾功 |
| **Procalcitonin (PCT)** | 免疫/ ⚠️ | **其他/** ✅ | 感染标志物，按"非特异炎症"归 其他 |
| **Vit D / B12 / 叶酸 / 铁蛋白** | 生化肝肾功/ ❌ | **其他/** ✅ | 营养/微量元素类，单独归 其他 |
| **HbA1c** | 生化肝肾功/ ⚠️ | **其他/** ✅（也可入 生化） | 血红蛋白糖化，最好独立成单 |
| **β-HCG** | 性激素/ ⚠️ | **性激素/** ✅（妊娠相关）或 **肿瘤标志物/**（GTN/绒癌） | 看文档语境 |
| **NT-proBNP** | 心脏标志物/ ✅ | 心脏标志物/ ✅ | 不要进 生化 |
| **PSA (前列腺特异)** | 性激素/ ❌ | **肿瘤标志物/** ✅ | 是肿瘤标志物 |

**多项目混合**：选项目数最多的子桶；并列时按表格顺序优先（血常规 > 生化 > 肿标 > ...）。

#### 03_分子病理

| 内容主体 | 子桶 |
|---|---|
| 基因突变位点 + VAF + MSI/TMB + 测序 | 基因检测/ |
| HPV 16/18/31/33/... 阳/阴 + PCR | HPV 分型/ |

#### 04_影像学子桶 vs 02 病理报告 边界

- 阴道镜检查报告（**仅观察+碘试验+拟诊**，无活检结果）→ `04_影像学/超声/`（视为光学检查）
- 阴道镜下活检 + ECC + 病理诊断 → `02_诊断与分期/病理报告/`
- 阴道镜下活检手术记录（无病理结论）→ `06_治疗记录/手术-内镜/`

### 步骤 3：09_患者补充 自动检测

文件名匹配以下模式 → 不进 02-08 而是进 09 子桶：

| 模式 | 09 子桶 |
|---|---|
| `*timeline*.txt` / `*timeline*.md` / `*治疗时间线*` / `*整理*.txt` (非医院文件名) | manual_timeline/ |
| `*wechat*` / `*微信*` (排除 `微信图片_*` — 那是医院报告的截图) | wechat/ |
| `*voice*` / `*语音*` / `*通话*` / `*转录*` | voice_transcripts/ |
| `*handwritten*` / `*手写*` (照片含"我"/"妈"/"爸"等亲属称谓) | handwritten/ |

**例外**：`微信图片_2026-02-20_175946.jpg` 这种纯时间戳的微信下载图片**不进 09**，按内容判桶（通常是医院 jpg 截图）。

### 步骤 4：confidence 处理

Layer 2 输出每个文件的 `classification_confidence ∈ [0, 1]`：
- ≥ 0.7 → 进目标子桶
- 0.5–0.7 → 进目标子桶 + flag `low_confidence_classification`（写进 readiness.warnings）
- < 0.5 → 进 `10_原始文件/未分类/` 等待人工 review

## 子桶 vs 桶级 — 何时 fallback 到桶级

如果 OCR 文本不足以判子桶（典型：PaddleOCR 失败 / 图片是手机拍的检验单一角，关键标题缺失）：

- `04_影像学/` 但无法判子类 → `04_影像学/其他/`
- `05_检验检查/` 但无法判子类 → `05_检验检查/其他/`
- `06_治疗记录/` 但无法判子类 → `06_治疗记录/支持治疗/`（最不会错）
- 无法判桶 → `10_原始文件/未分类/`

**禁止**写到桶根目录（如 `04_影像学/foo.jpg`）。每个文件必须落到子桶。

## 癌种特异子桶相关性表（v2.1 新增 — 用于 §4.8 空子桶清理决策）

骨架 §1.1 把所有子桶建出来，但很多子桶只对特定癌种相关。Layer 2 完成后，**空且无关**的子桶应在 §4.8 删掉。下表标注每个易"无关"子桶的相关癌种，**不在列内的癌种 + 子桶为空 → 删**。

| 子桶 | 主要相关癌种 | 反例（应删的癌种） |
|---|---|---|
| `03_分子病理/HPV 分型/` | 宫颈癌、口咽癌、肛管癌、外阴/阴道癌、部分头颈鳞癌 | 胰腺癌、肝癌、肺腺癌、乳腺癌、血液病 → 删 |
| `04_影像学/超声/` | 乳腺癌、甲状腺癌、肝胆胰癌（部分）、妇科癌（经阴道）、心动图（心脏标志物相关） | 大多数实体瘤如非乳/肝/妇 → 没装就删 |
| `05_检验检查/淋巴亚群/` | 血液病（淋巴瘤、白血病、MM、MDS）、骨髓移植后、HIV 合并癌 | 实体瘤一般无 → 删 |
| `05_检验检查/性激素/` | 妇科癌（卵巢/子宫内膜/乳腺）、前列腺癌、生殖细胞肿瘤、内分泌肿瘤 | 胰腺、肺、消化道 → 删 |
| `05_检验检查/心脏标志物/` | 蒽环类化疗后心毒性监测、HER2 靶向治疗（曲妥珠/T-DXd）心毒性、放射性心脏病 | 没用蒽环 / 没 HER2 靶向治疗 → 没装就删 |
| `06_治疗记录/放疗/` | 多数实体瘤都可能有；血液病（淋巴瘤 ISRT）也有 | 罕见情况无放疗 → 删 |
| `06_治疗记录/免疫治疗/` | 已 PD-L1+/MSI-H/TMB-H 接受 ICI 的患者 | MSS+CPS<1+TMB-L → 通常没 ICI → 没装就删 |
| `06_治疗记录/靶向/` | 有可靶向驱动突变的患者 | 无可靶向突变 → 没装就删 |

**对本次 PDAC 患者的应用示例**：
- ✅ 删: `HPV 分型/`（PDAC 不相关）、`淋巴亚群/`（实体瘤）、`性激素/`（无妇科适应症）、`心脏标志物/`（无蒽环）
- ✅ 保留即使空: `09_患者补充/*`（用户后续可补）、`10_原始文件/_duplicates/`、`10_原始文件/未分类/`、`11_诊断证明/`
- ✅ 保留有内容: 已用到的子桶
